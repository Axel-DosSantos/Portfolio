Mes Projets

Site web pour un artisan

Le but de ce projet était de concevoir et de développer un site web pour un artisan, j'ai donc choisi une chocolaterie artisanale.
Technologies Utilisées : HTML, CSS, PHP
Détails du Projet :
Conception Visuelle : J'ai élaboré une conception visuelle qui reflète l'élégance et la qualité des produits de la chocolaterie. Les couleurs chaudes et riches ont été utilisées pour évoquer une ambiance chocolatée et invitante.
Sections Principales :
Page d'accueil : Qui montre les couleurs du site et explique l'origine de la chocolaterie.
Page Équipe : Qui présente l'équipe ainsi que leur profil LinkedIn.
Page Réalisations : Qui présente tous les produits de l'enseigne avec une page dédiée pour chaque produit.
Page Partenaires : Qui présente tous les partenaires, chaque logo renvoyant au site du partenaire.
Page Tarifs : Avec différentes quantités de produits et leurs prix.
Page Contact : Une section permettant aux clients de poser des questions, de faire des commentaires, avec des options de contact via un formulaire de contact.
Résultats :
Ce projet avait surtout pour but d'utiliser les connaissances fraîchement acquises et il a servi de base pour tous mes projets futurs.
